# PFA-IDS-IoT-LLMs-Deep-Learning
 système capable d’analyser les flux de données des dispositifs IoT d’un réseau domestique ou industriel et de détecter automatiquement les comportements suspects ou attaques (malwares, botnets, spoofing). Le projet combine Deep Learning (GRU/LSTM) pour l’analyse séquentielle et LLMs pour générer des alertes compréhensibles.
